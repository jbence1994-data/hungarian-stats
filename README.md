# Hungarian Stats

Statistical data from Hungary

[![Continuous Integration](https://github.com/jbence1994-data/hungarian-stats/actions/workflows/build.yaml/badge.svg)](https://github.com/jbence1994-data/hungarian-stats/actions/workflows/build.yaml)

## Prerequisites

Make sure you have the following tools installed:

- [Bun 1.2+](https://bun.com/get)

## Run project locally

Before running the application, copy `.env.example` to `.env` and fill in your values.

### Install project dependencies

```bash
bun install
```

### Start the application

```bash
bun run dev
```

- Dev server: http://localhost:3000
